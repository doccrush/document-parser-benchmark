# Benchmark Documents


This dataset contains challenging PDF documents used for evaluating document parsing tools.


## Categories


| Category | Purpose |
|---|---|
| Academic | Multi-column layouts, equations, citations |
| Financial | Tables, charts, structured data |
| Technical | Headings, code blocks, images |
| Scanned | OCR capability |
| Presentation | Visual layouts, bullet points |


## Evaluation Focus

Each document is evaluated on:

- Reading order
- Heading structure
- Table extraction
- OCR quality
- Markdown cleanliness
- RAG readiness
