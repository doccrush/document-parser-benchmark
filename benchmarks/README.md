# Benchmark Dataset

This dataset contains real-world documents used to evaluate document parsing tools.

## Categories

### 1. Standard PDFs

Documents with normal text layout.

Examples:

- Articles
- Reports
- Documentation


### 2. Tables

Documents containing:

- Financial tables
- Data sheets
- Complex layouts


### 3. Scanned Documents

Documents requiring OCR.

Examples:

- Scanned books
- Images
- Historical documents


### 4. Multi-column PDFs

Documents with:

- Academic papers
- Magazine layouts
- Complex reading order

## Evaluation Goal

The benchmark focuses on:

- Reading order
- Structure preservation
- Markdown quality
- Table extraction
- RAG usability
