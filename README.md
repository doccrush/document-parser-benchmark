# Document Parser Benchmark

A benchmark comparing open-source document parsing tools for AI and RAG pipelines.

## Tested tools

- Microsoft MarkItDown
- Marker
- Docling
- MinerU
- PaddleOCR

## Why?

Poor document parsing leads to:

- broken chunks
- missing tables
- bad retrieval results

## Goal

Find the best way to convert documents into clean Markdown for LLM applications.
